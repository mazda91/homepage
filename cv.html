Internship
2014 PlanIt: A Crowdsourcing Approach for Learning to Plan Paths from Large Scale Preference
Feedback.
Mentor: Prof. Ashutosh Saxena (Cornell University)
PlanIt extends the principle of factoring in user preferences in path planning by allowing us
to obtain these preferences on a large scale.
Extended the existing cost function based 2-D navigation algorithm to work with high di-
mensional arm manipulation scenarios. Also worked on visualizing these learned costs as 3-D
heatmaps. Integrated all these features into the online PlanIt system.
Research
2013 MTBA: Matlab Toolbox for Biclustering Analysis
Mentor: Prof. Nishchal K. Verma (IIT Kanpur)
Developed a new Matlab toolbox, MTBA, designed to perform a variety of biclustering al-
gorithms under a common user interface while providing additional facilities for data pre-
processing, visualization, and validation. is allows the user to compare biclustering results
from diﬀerent algorithms and choose the approach that best ﬁts their particular scenario.
2012 Feature Level Analysis
Mentor: Prof. Nishchal K. Verma (IIT Kanpur)
Undertook a case study of acoustic and vibrational features from diﬀerent working states of
an air compressor. Graphically analyzed these features to derive the best feature set. Built an
SVM model based upon these features and got comparable results to the standard PCA based
SVM model.
2012 Condition Based Monitoring of Air Compressors and Motors using Acoustic Data
Mentor: Prof. Nishchal K. Verma (IIT Kanpur)
Developed a smart phone application, to learn diﬀerent fault states of an industrial air com-
pressor. Tested the application to recognize the fault state in real time as the air compressor
was running. It has performed very well with classiﬁcation accuracies above 93.73%. Similar
application and model with some minor changes in speciﬁcations can be used for acoustic
pattern recognition in wide range of ﬁelds; especially in industry.
